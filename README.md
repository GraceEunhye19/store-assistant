Store Assistant App

Store Assistant is an Android app built to simplify shopping and product management.  
It provides an intuitive UI for browsing, viewing, and managing store items efficiently.

---

Features
- Modern UI built with Jetpack Compose
- Browse products in a carousel
- View detailed item information
- Smooth and responsive user experience

---

Tech Stack
- Language: Kotlin  
- Framework: Android Jetpack Compose  
- Architecture: MVVM  
- Tools: Android Studio, Coil for image loading

---

Installation
1. Download the latest APK from the link below.  
2. Open the file on your Android phone.
